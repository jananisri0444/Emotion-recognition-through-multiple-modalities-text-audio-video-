# Emotion-recognition-through-multiple-modalities-text-audio-video
Individual models for emotion recognition are increasingly relevant in fields like human
computer interaction, mental health monitoring, and social robotics. Utilizing the 
IEMOCAP dataset, this study emphasizes the development of distinct models for audio, 
text, and motion analysis.

For audio processing, a hybrid architecture integrates Convolutional Neural Networks 
(CNN), Multi-Head Attention, and Gated Recurrent Units (GRU) to capture temporal 
dependencies and contextual information, achieving an accuracy of 81%. The text 
analysis employs a CNN architecture that mimics Temporal Convolutional Networks 
(TCN) for efficient feature extraction, resulting in a high accuracy of 94%. Motion analysis 
utilizes Spatio-Temporal Graph Convolutional Networks (ST-GCN), which focus on visual 
cues, yielding an accuracy of 63%. 

The methodology encompasses data preprocessing, feature extraction, and model 
training, culminating in a comprehensive evaluation of performance metrics. Results 
indicate significant improvements in accuracy and robustness in emotion classification 
tasks when compared to traditional approaches. These findings highlight the 
effectiveness of distinct model architectures in enhancing emotion recognition 
capabilities, setting the stage for future advancements in the field.
